## How to run this Application

Change env.example -> .env

Open your favorite Terminal and run these commands.

Run Tab:

```sh
php artisan key:generate
```

```sh
php artisan config:cache
```

```sh
composer install
```

```sh
npm install
```

```sh
php artisan serve 
```
