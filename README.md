## How to run this Application

Change env.example -> .env

in .env 
```sh
DB_CONNECTION=mongodb
DB_HOST=localhost
DB_PORT=27017
DB_DATABASE= // name of ur database
DB_USERNAME=
DB_PASSWORD=
```

Open your favorite Terminal and run these commands.

Run Tab:

```sh
composer install
```

```sh
npm install
```

```sh
php artisan key:generate
```

```sh
php artisan config:cache
```


```sh
php artisan serve 
```
